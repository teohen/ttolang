# ttolang
A interpreted, dinamic typed, and really cool language created by me.

## features of ttolang
- [x] variables
- [x] integers and bools
- [x] arithmetic expressions
- [x] built-in functions
- [x] functions: first class and higher order 
- [x] closures 
- [x] string DS 
- [x] List DS
