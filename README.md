## Repetição

Estrutura de repetição que cria o seu próprio escopo interno e também permite acessar aos valores do escopo "pai".

Palavra reservada: `REPETE`

Estrutura:

repete ([`ATRIBUIÇÃO`](#atribuição) até [`CONDIÇÃO`](#condição)) {

  [`EXPRESSÃO`](#expressão)
  
}
