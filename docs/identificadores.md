## Identificadores
Em ttolang, identificadores são utilizados para nomear [variáveis](variaveis.md) e [procedimentos](procedimentos.md) criados. 
Para nomear esses itens, deve-se seguir a seguintes regras:
1. Utilizar o padrão de nomeclatura conhecido como "snake_case".
2. Somente utillizar letras para criar identificadores. 
3. Não é permitido repetir nomes dados a identificadores diferentes. 
4. Não pode conter espaços.
5. Não pode ser uma [palavra reservada](palavras-reservadas.md) da linguagem ttolang. 

Os identificadores podem ser utilizados em conjunto com o símbolo de [atribuição](atribuição.md) (<-) para nomear os valores que estão (ou serão) definidos no escopo de valores.
